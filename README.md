
# MDIC Creative Writing Club Portal 

A full stack web platform for the Creative Writing Club of 
Manarat Dhaka International School and COllege (MDIC).
Built entirely by students, for students.

## 🌟 What Is This?

A digital creative portal where students can submit their 
written works like articles, essays, fiction, blogs and more. 
Teachers and editors can review, approve, and highlight 
the best pieces every month.

## ✨ Features

- 📖 Public magazine-style homepage
- 🗂️ Category browser (Articles, Essays, Blogs, Fiction, 
     Features, Biographies, Reviews)
-  Secure student submission desk
- 🏆 Editor's Choice and Writer of the Month system
-  Archive of past newsletters and works
-  Teacher and editor review dashboard

## 🛠️ Built With

- **Frontend** — Next.js + Tailwind CSS
- **Database** — Supabase (PostgreSQL)
- **Hosting** — Vercel
- **Design** — Figma
